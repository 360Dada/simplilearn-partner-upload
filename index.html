📖 Detailed guidelines
🎞Format: .mp4 only. No other formats accepted.
📐Resolution: Minimum 1080p (1920×1080). Clear audio, no background noise.
⚠️Name every file following the video naming structure before uploading. See the Naming tab.
# One folder per lesson, videos at root level
📁 Lesson_01_Introduction/
  1.01_What_is_the_Course.mp4
  1.02_About_the_Instructor.mp4
📁 Lesson_02_Getting_Started/
  2.01_Environment_Setup.mp4
⚠️
The extension must be .en.vtt — not .vtt
The .en. denotes English language and is required by Simplilearn's platform. Files submitted as .vtt will be returned for correction.
📄Format: .en.vtt only. Neither .vtt nor .srt accepted.
📁Location: Inside a subfolder named transcript (lowercase, no prefix) within each lesson folder.
🔗Filename: Identical to the video filename — replace .mp4 with .en.vtt. One caption file per video.
# Videos at lesson root, captions in transcript subfolder
📁 Lesson_01_Introduction/
  1.01_What_is_the_Course.mp4
  1.02_About_the_Instructor.mp4
  📁 transcript/
    1.01_What_is_the_Course.en.vtt
    1.02_About_the_Instructor.en.vtt
📋Format: .csv using the Simplilearn template. No other format accepted.
❓Questions: MCQ with 4 options each. Mark correct answer with 1, others with 0.
💡Include a clear explanation for every question. Use underscores in topic names — no spaces.
Sr.No | Topic | QuestionType | Questions | Options | Correct Answer | Explanation
1 | AI_Literacy | MCQ | What is AI? | Option A | 0 | Explanation...
    |             |     |            | Option B | 0 | Explanation...
    |             |     |            | Option C | 1 | Explanation...
    |             |     |            | Option D | 0 | Explanation...
⏱Duration format is HH:MM:SS. Every topic must have a duration. Do not leave blank.
🔢Topic numbers must match your video filenames exactly — e.g. Topic 1.01 → file named 1.01_Topic_Name.mp4.
📊Fill in: Course Name, Lesson No., Lesson Name, Topic No., Topic Title, Duration.
Course Name    | Lesson No. | Lesson Name  | Topic No. | Topics                  | Duration
Azure AI Studio | 1            | Introduction | 1.01       | What is Azure AI Studio     | 00:05:18
                 |              |              | 1.02       | Features of Azure AI Studio | 00:08:45
📐Formula: [Topic_Number]_[Topic_Title_With_Underscores].mp4
🔤Underscores between words. No spaces, no special characters, no suffixes like FINAL or v2.
🔢Topic number uses a dot: 1.01, 2.03, 5.11
✓ 1.01_What_is_Azure_AI_Studio.mp4
✓ 2.03_Resources_and_Jupyter_Notebook.mp4
✓ 5.11_Analyze_and_Query_Data_with_LLMs.mp4

✗ What is Azure AI Studio.mp4 (spaces not allowed)
✗ Lesson1_Topic1.mp4 (wrong format)
✗ 1.01_What_is_Azure_AI_Studio_FINAL.mp4 (no suffixes)
📥Downloadable resources — code files, datasets, cheat sheets, project files. Include only if used or mentioned in a video.
📚Supporting materials — slides, worksheets, reference PDFs. Include if referenced in your videos.
🏷Name resources with the topic number prefix where relevant: 3.01_Starter_Code.zip
